What is Cowrie-web
*****************************************

Cowrie-web is a simple php site to get info from Cowrie stored in mysql.
It requires `Cowrie <http://github.com/cowrie/cowrie/>`_ to be installed and 
`Set up to store data in mysql <https://cowrie.readthedocs.io/en/latest/sql/README.html#how-to-send-cowrie-output-to-a-mysql-database>`_

`Cowrie-Web <http://github.com/dr3as/cowrie-web/>`_ is maintained by Me.
